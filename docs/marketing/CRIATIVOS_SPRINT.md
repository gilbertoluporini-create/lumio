## D. Naming convention de UTMs (não esquecer)

Padrão pra usar em TODA campanha paga + orgânica:

```
?utm_source={canal}&utm_medium={tipo}&utm_campaign={produto}_{conceito}&utm_content={variacao}
```

Exemplos prontos:
- Meta IG Reels conceito A var 1: `?utm_source=meta&utm_medium=reels&utm_campaign=lumio_emocional&utm_content=a1`
- TikTok orgânico hook 1: `?utm_source=tiktok&utm_medium=organic&utm_campaign=lumio_pov&utm_content=h1`
- Google exact `transcrever aula`: `?utm_source=google&utm_medium=cpc&utm_campaign=lumio_intent&utm_content=transcrever_aula_exact`

**Cole no Bitly/short.io** se a URL ficar feia no anúncio. UTMs sobrevivem o redirect.

## E. Configuração Meta Aggregated Event Measurement (AEM)

iOS 17+ limita: **8 eventos prioritários por domain**, restante é filtrado. Pra Lumio, priorizar nesta ordem:

| # | Evento | Razão |
|---|--------|-------|
| 1 | `Purchase` | Maior valor — otimizar conversão paga |
| 2 | `Subscribe` | Trigger no checkout.session.completed |
| 3 | `CompleteRegistration` | Signup completo (sign_up custom) |
| 4 | `InitiateCheckout` | Begin checkout (Analytics.beginCheckout) |
| 5 | `Lead` | Magic link request, embaixador form |
| 6 | `Login` | Returning user — pra retargeting |
| 7 | `ViewContent` | View item específico — granular |
| 8 | `PageView` | Catch-all |

**Como configurar:**
1. business.facebook.com → Events Manager → Pixel "Lumio"
2. Aggregated Event Measurement (tab no menu)
3. Domain `lumioapp.net` → Manage Events → arrastar nessa ordem
4. Salvar — leva ~24h pra propagar

**Pré-requisito:** domain `lumioapp.net` precisa estar **Verified** no Business Settings → Brand Safety → Domains. Verificação por meta tag já tá pronta pra adicionar (próximo doc).
